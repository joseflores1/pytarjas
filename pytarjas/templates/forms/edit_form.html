Forms are the foundation of data collection in the field, so this interface must be both powerful and intuitive. It supports editing form metadata AND managing the dynamic list of questions. FEATURES: - Edit form metadata (name, description, type, active status) - Dynamic question management (add, edit, reorder, delete questions) - Drag-and-drop question reordering (optional enhancement) - Real-time validation - Touch-optimized for tablets - Save/cancel with confirmation on unsaved changes - Question type selector with appropriate options WORKFLOW CONTEXT: Forms define the structure of data collection: 1.
